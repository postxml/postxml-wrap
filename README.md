# postxml-wrap
[![npm version][npm-image]][npm-url]

> [PostXML] plugin wrap with emmet abbreviation. Support only tags and classes.

## Installation
`npm i postxml- --save-dev`

## Usage
[PostXML]

## Example

### Input
```html
<div wrap=".g-section.bg"><a></a></div>
```

### Output
```html
<div class="g-section bg"><div><a></a></div></div>
```

## Options

### attr
Type: `String`<br>
Default: `wrap`
